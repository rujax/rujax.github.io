开始之前

Tip

也可以根据自己电脑使用的 CPU 指令集点击 Portable 下方的链接下载自解压便携版本的 Git。

搭建 UI

改造顶部布局

首先打开 mainwindow.h 文件，在 private 下添加单行编辑控件 QLineEdit 和按钮控件 QPushButton ：

...

private:
    QLineEdit *_feedEdit;
    QPushButton *_subscribeButton;

...

添加后，编辑器会提示这2个类型是未知的。
在文件顶部添加这2个类的声明即可。

...

#include <QMainWindow>

// 新增
class QLineEdit;
class QPushButton;

class MainWindow : public QMainWindow

...

接着打开 mainwindow.cpp 文件，在文件顶部添加上述2个控件的库 #include 之后，在 _buildUI() 方法中做如下调整：

...

#include <QBoxLayout>
#include <QLabel>
// 新增
#include <QLineEdit>
#include <QPushButton>

...

void MainWindow::_buildUI()
{
    // 调整前
    QLabel *topLabel = new QLabel("top");
    topLabel->setAlignment(Qt::AlignCenter); // 设置文本居中
    topLabel->setStyleSheet("background-color: Aqua; font-size: 24px"); // 设置文本样式

    QHBoxLayout *topLayout = new QHBoxLayout;
    topLayout->addWidget(topLabel);

    // 调整后
    int spacing = 20;

    _feedEdit = new QLineEdit();
    _feedEdit->setPlaceholderText("请输入 RSS 链接");

    _subscribeButton = new QPushButton("订阅");

    QHBoxLayout *subscribeLayout = new QHBoxLayout();
    subscribeLayout->setSpacing(spacing);
    subscribeLayout->setContentsMargins(spacing, spacing, spacing, spacing / 2);
    subscribeLayout->addWidget(_feedEdit);
    subscribeLayout->addWidget(_subscribeButton);

    ...

    // 调整前
    mainLayout->addLayout(topLayout, 2);

    // 调整后
    mainLayout->addLayout(subscribeLayout);

...

简单描述下代码：

  1. 初始化间距变量 spacing 并设置 20
  2. 初始化单行编辑控件 _feedEdit，并调用方法 setPlaceholderText() 设置占位文本
  3. 初始化按钮控件 _subscribeButton 并设置文本
  4. 初始化水平盒布局 subscribeLayout
    • 调用方法 setSpacing() 设置组件间的间距
    • 调用方法 setContentsMargins 设置内容到布局四个边的边距
  5. 将 2、3 创建的组件添加到 4 的布局中
  6. 在主布局 mainLayout 中添加布局 subscribeLayout

改造底部布局

改造前的准备

调整代码之前，还需要做一些准备：

  1. 在代码中获取到应用的版本号
  2. 添加设置按钮的图标

先做第1点。

获取版本号

在左侧文件结构右键 RSSReader 后点击 添加新文件，依次选择 概要 → Empty File 后点击 选择：

在新弹窗中填写文件名 project.h.in 后点击 下一步 后再点击完成，完成文件创建：

接下来打开 CMakeLists.txt 文件，做如下调整：

cmake_minimum_required(VERSION 3.19)
project(RSSReader VERSION 0.0.1 LANGUAGES CXX)

# 新增
configure_file(
    project.h.in # 输入文件
    project.h    # 输出文件
    @ONLY        # 仅替换 @VAR@ 形式的变量
)

...

set(HEADERS
    mainwindow.h
    project.h # 新增
)

qt_add_executable(RSSReader
    WIN32 MACOSX_BUNDLE
    ${SOURCES}
    ${HEADERS}
)

# 新增
target_include_directories(RSSReader PRIVATE ${CMAKE_CURRENT_BINARY_DIR})

...

Tip

qt_add_executable() 命令中删除掉 project.h.in。

再打开新建的 project.h.in，填入以下内容：

#ifndef PROJECT_H
#define PROJECT_H

#define PROJECT_VERSION "@PROJECT_VERSION@"

#endif // PROJECT_H

为了能在代码中获取到应用版本号，这里使用了命令 configure_file 和 target_include_directories 。

configure_file 的作用是将输入文件 project.h.in 复制为输出文件 project.h，同时将输入文件中引用的 环境变量 的值进行 替换 。

因为上一篇 CMakeLists.txt 文件中， project() 命令中添加了 VERSION 选项，它设置了 PROJECT_VERSION 变量。
所以，project.h.in 文件中引用的变量 "@PROJECT_VERSION@" 可以被替换为 CMakeLists.txt 文件中设置的值，并且定义为宏 PROJECT_VERSION。

修改完成后，保存 CMakeLists.txt 文件，此时项目会重新编译，同时左侧文件结构也出现了 project.h.in 和 project.h 的身影：

Note

另外还有一点要说的是：
project.h 能被其他文件 #include，多亏了命令 target_include_directories 将其添加到项目中。

添加图标

执行新建文件操作，依次选择 Qt → Qt Resource File：

填入文件名 project 后点击 下一步，再点击 完成：

再次打开 CMakeLists.txt 文件，进行调整：

...

configure_file(
    project.h.in # 输入文件
    project.h    # 输出文件
    @ONLY        # 仅替换 @VAR@ 形式的变量
)

# 新增
set(CMAKE_AUTORCC ON)

...

set(HEADERS
    mainwindow.h
    project.h
)

# 新增
set(RESOURCES
    project.qrc
)

qt_add_executable(RSSReader
    WIN32 MACOSX_BUNDLE
    ${SOURCES}
    ${HEADERS}
    ${RESOURCES} # 新增
)

...

首先通过 set(CMAKE_AUTORCC ON) 启用 AUTORCC 处理 .qrc 文件，
然后设置一个变量存放资源文件，并在 qt_add_executable() 中使用。

Tip

qt_add_executable() 命令中删除掉 project.qrc。

此时，左侧文件结构里可以看到 project.qrc 文件的身影。
找一个设置按钮的 SVG 文件，并命名为 config.svg。
右键文件，选择 添加现有文件，在弹窗中选择 config.svg 文件：

左侧文件结构可以看到 config.svg 文件，同时得知了它的访问路径为 /config.svg。

继续改造底部布局

2点准备都完成了，继续改造底部布局。

打开 mainwindow.cpp 文件，继续调整：

...

#include <QPushButton>

#include "project.h" // 新增

MainWindow::MainWindow(QWidget *parent) : QMainWindow(parent)

...

void MainWindow::_buildUI()
{
    ...

    // 修改前
    QLabel *bottomLabel = new QLabel("bottom");
    bottomLabel->setAlignment(Qt::AlignCenter);
    bottomLabel->setStyleSheet("background-color: SpringGreen; font-size: 24px");

    QHBoxLayout *bottomLayout = new QHBoxLayout;
    bottomLayout->addWidget(bottomLabel);

    // 修改后
    QLabel *versionLabel = new QLabel(QString("版本: %1").arg(PROJECT_VERSION));
    versionLabel->setStyleSheet("font-size: 14px");

    QPushButton *configButton = new QPushButton(QIcon(":/config.svg"), "");
    configButton->setIconSize(QSize(24, 24));

    QHBoxLayout *informationLayout = new QHBoxLayout();
    informationLayout->setSpacing(spacing);
    informationLayout->setContentsMargins(spacing / 2, spacing / 2, spacing / 2, spacing / 2);
    informationLayout->addWidget(versionLabel);
    informationLayout->addStretch();
    informationLayout->addWidget(configButton);

    ...

    // 修改前
    mainLayout->addLayout(bottomLayout, 1);

    // 修改后
    mainLayout->addLayout(informationLayout);

    ...
}

简单描述下代码：

  1. 初始化版本号标签 versionLabel，并设置版本号内容
  2. 初始化设置按钮 configButton，并设置图标，再调用方法 setIconSize() 设置图标大小
  3. 初始化水平盒布局 informationLayout，依次添加 1、2 创建的组件，并且在组件之间调用方法 addStretch() 添加一个弹簧撑开左右的组件
  4. 在主布局 mainLayout 中添加布局 informationLayout

改造中间布局

我打算使用网格布局组件 QGridLayout 和滚动区域组件 QScrollArea 配合搭建中间布局的 UI。

先打开 mainwindow.h 文件，进行调整：

...

class QLineEdit;
class QPushButton;
class QGridLayout; // 新增

class MainWindow : public QMainWindow
{
    ...

private:
    QLineEdit *_feedEdit;
    QPushButton *_subscribeButton;
    QGridLayout *_feedsLayout; // 新增

    ...

再打开 mainwindow.cpp 文件，进行调整：

...

#include <QPushButton>
#include <QScrollArea> // 新增

...

void MainWindow::_buildUI()
{
    ...

    // 修改前
    QLabel *middleLabel = new QLabel("middle");
    middleLabel->setAlignment(Qt::AlignCenter);
    middleLabel->setStyleSheet("background-color: Violet; font-size: 24px");

    QHBoxLayout *middleLayout = new QHBoxLayout;
    middleLayout->addWidget(middleLabel);

    // 修改后
    _feedsLayout = new QGridLayout();
    _feedsLayout->setSpacing(spacing);
    _feedsLayout->setContentsMargins(spacing, spacing / 2, spacing, 0);

    int row = 5;
    int col = 3;

    for (int r = 0; r < row; ++r)
    {
        for (int c = 0; c < col; ++c)
        {
            QLabel *feedLabel = new QLabel(QString::number(r * col + c + 1));
            feedLabel->setAlignment(Qt::AlignCenter);
            feedLabel->setStyleSheet("background-color: Violet; font-size: 24px");
            feedLabel->setMinimumSize(QSize(240, 180));

            _feedsLayout->addWidget(feedLabel, r, c);
        }
    }

    QWidget *feedsContainer = new QWidget;
    feedsContainer->setLayout(_feedsLayout);

    QScrollArea *feedsScrollArea = new QScrollArea();
    feedsScrollArea->setHorizontalScrollBarPolicy(Qt::ScrollBarAlwaysOff);
    feedsScrollArea->setFrameShape(QFrame::NoFrame);
    feedsScrollArea->setWidget(feedsContainer);
    
    ...

    // 修改前
    mainLayout->addLayout(middleLayout, 9);

    // 修改后
    mainLayout->addWidget(feedsScrollArea);

    ...
}

简单描述下代码：

  1. 初始化网格布局 _feedsLayout
  2. 临时生成5行3列标签内容用来查看效果
  3. 初始化滚动区域的容器组件 feedsContainer，并调用方法 setLayout() 将 _feedsLayout 设置为其布局
  4. 初始化滚动区域组件 feedsScrollArea
    • 调用方法 setHorizontalScrollBarPolicy() 设置横向滚动条不启用
    • 调用方法 setFrameShape() 隐藏滚动条的背景
