WARNING

开始之前

在开始今天的 Qt 应用开发之前，先将上一篇的代码变动提交到仓库里。

检查文件差异

进入项目文件夹后执行 git status 可以查看到当前的文件状态：

可以看到：config.svg、project.h.in 和 project.qrc 被列为新文件，
而 CMakeLists.txt、mainwindow.cpp 和 mainwindow.h 被列为已修改的状态。

通过 git diff 命令可以查看已修改文件的差异。

Note

查看差异的方式有很多种，比如：Qt Creator 内置的 Git 工具和 Visual Studio Code 的 Git 工具以及相关插件。

确认文件差异无误后，依次使用 git add . 和 git commit -m <message> 提交代码。

添加样式

在 Qt 应用中添加样式除了对继承自 QWidget 的组件使用 setStyleSheet() 方法，
还可以通过添加文件扩展名为 .qss 的样式文件，调用 QApplication 的 setStyleSheet() 方法应用到整个项目之中。

添加 .qss 文件

右键 project.qrc 文件，选择 添加新文件...，在弹窗中依次选择 概要 → Empty File 后，点击 选择…：

文件名填入：project.qss 后，点击 下一步，在新的弹窗中点击 完成：

此时，可以看到 project.qss 文件已经在左侧文件结构 Resources → project.qrc → / 的路径之下了。

Qt Style Sheets

Qt Style Sheets 是 Qt 应用中的 Cascading Style Sheets(CSS) ，
它的 语法 深受 CSS 启发。

官方文档 Qt Style Sheets Reference 和 Qt Style Sheets Examples 都是非常好的学习资料。

调整代码

main.cpp

添加如下高亮代码：

#include "mainwindow.h"

#include <QApplication>
#include <QFile>

int main(int argc, char *argv[])
{
    QApplication a(argc, argv);
    QFile qss(":/project.qss");

    if (qss.open(QFile::ReadOnly))
    {
        QString style = qss.readAll();
        a.setStyleSheet(style);

        qss.close();
    }
    else
    {
        qWarning() << "Can't open project.qss";
    }

    MainWindow w;
    ...
}

简单描述下代码：

  1. 调用 QFile 的构造函数生成实例 qss
  2. 再调用 open() 方法以只读模式打开文件
  3. 调用 readAll() 方法读取文件内容并赋值给字符串
  4. 调用 setStyleSheet() 方法应用样式
  5. 调用 close() 方法关闭文件
  6. 如果打开文件失败，则调用宏 qWarning() 打印失败信息

运行应用，查看底部应用输出，没有输出 Can't open project.qss 则代表加载成功。

mainwindow.cpp

打开 mainwindow.cpp 文件，搜索关键字 setStyleSheet，可以看到代码有 2 处使用。

第一处 

...

void MainWindow::_buildUI()
{
    ...

    for (int r = 0; r < row; ++r)
    {
        for (int c = 0; c < col; ++c)
        {
            QLabel *feedLabel = new QLabel(QString::number(r * col + c + 1));

            // 修改前
            feedLabel->setAlignment(Qt::AlignCenter);
            feedLabel->setStyleSheet("background-color: Violet; font-size: 24px");

            // 修改后
            feedLabel->setProperty("class", "feed-label");
            feedLabel->setAlignment(Qt::AlignCenter);

            ...
        }
    }

    ...
}

通过调用 setProperty() 方法，
给每一个生成的 feedLabel 添加属性 class，属性值为 feed-label。

设置属性 class 的作用类似 CSS 的 类选择器 ， 指定这类标签都拥有这个属性。

接着，打开 project.qss，填入以下内容：

QLabel.feed-label {
    background-color: Violet;
    font-size: 24px;
}

运行应用，可以看到窗口中的内容与原先一致。

第二处 

...

void MainWindow::_buildUI()
{
    ...

    QLabel *versionLabel = new QLabel(QString("版本: %1").arg(PROJECT_VERSION));

    // 修改前
    versionLabel->setStyleSheet("font-size: 14px");

    // 修改后
    versionLabel->setObjectName("version-label");

    ...
}

通过调用 setObjectName() 方法为 versionLabel 指定名称，
在 qss 中则类似 CSS 的 ID 选择器 的效果。

打开 project.qss，填入以下内容：

QLabel#version-label {
    font-size: 14px;
}

运行应用，与之前样式一致即可。

美化样式

窗口背景色

继续修改 mainwindow.cpp 文件：

...

void MainWindow::_buildUI()
{
    ...

    QWidget *feedsContainer = new QWidget;
    feedsContainer->setObjectName("feeds-container");
    feedsContainer->setLayout(_feedsLayout);

    ...

    QWidget *container = new QWidget(this);
    container->setObjectName("container");
    container->setLayout(mainLayout);

    ...
}

继续修改 project.qss 文件，在顶部添加：

QWidget#container, QWidget#feeds-container {
    background-color: #fcfcfc;
}

美化顶部订阅布局

美化单行编辑组件和订阅按钮

继续修改 mainwindow.cpp 文件：

...

void MainWindow::_buildUI()
{
    ...

    _feedEdit = new QLineEdit();
    _feedEdit->setObjectName("feed-edit");
    _feedEdit->setPlaceholderText("请输入 RSS 链接");

    _subscribeButton = new QPushButton("订阅");
    _subscribeButton->setObjectName("subscribe-button");

    ...
}

继续修改 project.qss 文件：

...

QLineEdit#feed-edit {
    height: 60px;
    padding: 5px 10px;
    border: 1px solid gray;
    border-radius: 10px;
    font-size: 24px;
}

QPushButton#subscribe-button {
    width: 180px;
    height: 60px;
    padding: 5px;
    background-color: #ff9632;
    border-radius: 10px;
    font-size: 20px;
    font-weight: bold;
    color: white;
}

QPushButton#subscribe-button:hover {
    background-color: #ee802f;
}

QPushButton#subscribe-button:pressed {
    background-color: #ff9632;
}

为顶部订阅布局添加下边框

这里通过 QFrame 组件实现一条边框，继续修改 mainwindow.cpp 文件：

...

void MainWindow::_buildUI()
{
    ...

    subscribeLayout->addWidget(_subscribeButton);

    QFrame *subscribeLine = new QFrame();
    subscribeLine->setObjectName("subscribe-line");
    subscribeLine->setFrameShape(QFrame::HLine);
    subscribeLine->setContentsMargins(spacing + 5, 0, spacing + 5, 0);

    ...

    mainLayout->addLayout(subscribeLayout);
    mainLayout->addWidget(subscribeLine);
    mainLayout->addWidget(feedsScrollArea);

    ...
}

继续修改 project.qss 文件：

...

QFrame#subscribe-line {
    color: lightgray;
}
